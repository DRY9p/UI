```bash
  cd test/dummy && ./bin/dev
```
## Find me by url
```
  http://127.0.0.1:3001/ui
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem "ui"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ui
```
